Parameters: - providers (required) — JSON string of [{slug, label}] for enabled providers (the model catalog itself is fetched dynamically on first popover open via ``automation:agent_models``) - field_name_model (default — name of the hidden input mirroring the selected "agent_model") ``provider:model`` spec.
