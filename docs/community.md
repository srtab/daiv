# Community

DAIV is open-source and contributions are welcome — code, documentation, testing, or feedback.

## Contributing

For contribution guidelines, code style, testing requirements, and the pull request process, see [CONTRIBUTING.md](https://github.com/srtab/daiv/blob/main/CONTRIBUTING.md).

## Reporting issues

Use [GitHub Issues](https://github.com/srtab/daiv/issues) for bug reports, feature requests, and questions. When reporting:

- Search existing issues first
- Include environment details, steps to reproduce, and any relevant logs or screenshots
- For security issues, follow responsible disclosure as outlined in CONTRIBUTING.md
